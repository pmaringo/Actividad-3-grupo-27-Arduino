# UNIR---Actividad-3-Grupo-27-
Diseño e implementación del firmware de un ascensor inteligente para un edificio de cinco plantas, plataforma Arduino UNO. Funcionamiento en torno a varias técnicas de control: Máquina de estados finita (FSM),algoritmo SCAN (paradas),un controlador PID (temperatura),un control con histeresis (iluminación) y un controlador difuso (velocidad).
